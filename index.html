Cinder.js is a Lightweight & Extensible Layout Library

With its cruft-free mix of speed and pixel perfection, Cinder.js ensures your blocks remain beautiful, always.

Cinder.js supports multiple layout algorithms, including masonry, and developers can easily define their own. The library is available in vanilla JS and as a jQuery plugin that is also compatible with cash. Both versions are under 2KB when minified and gzipped.

Cinder.js Demo

Content

Cinder.js works with fixed ratio content like images, dynamic content like text, or a combination of the two.

Layout

Cinder.js includes a number of algorithms, and it is easy for developers to define their own.

Animation

Cinder.js is designed to work well with CSS transitions. However, you can also use an external library to control animations for older browsers.
